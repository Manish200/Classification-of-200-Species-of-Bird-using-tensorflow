This project is helps in classification of 200 bird species using tensorflow. tensorflow is the ML framework that Google created. It is used to design, build, and train deep learning models. 

In this Load-Test-Data.py is used for testing the model in this we have taken 70:30 ratio between training nd testing data.
Load-Train-Data.py is used in training the model. 

You can download the dataset from kaggle - 

http://www.vision.caltech.edu/visipedia/CUB-200-2011.html
